5 Core Navigation Links

Every page must have these 5 links in this order:

Order Label URL Purpose Mobile Icon
1 Home /home.html Role dashboard, overview 🏠
2 Library /library.html Rights, laws, research 📚
3 Office /office.html Documents, vault, communication 🏢
4 Tools /tools.html Calculators, generators 🔧
5 Help /help.html Support, FAQ, crisis resources 🆘

Active State:

The current page's nav link gets class="active" styling (highlighted)

Role-Specific Navigation

Role Theme Home URL Special Nav Items
Public 🌊 Ocean /public/welcome.html Sign In / Get Started (buttons)

Sub-Page Navigation Patterns

Office Sub-Pages (all under /office/):

  • /office/vault.html - Document upload & management
  • /office/inbox.html - Communication center
  • /office/timeline.html - Journal/timeline view
  • /office/delivery.html - Send documents out
  • /office/signer.html - Sign PDFs (desktop only)

Library Sub-Pages:

  • /library/rights.html - Rights guides
  • /library/laws.html - Law database search
  • /library/forms.html - Form templates

Tools Sub-Pages:

  • /tools/calculators.html - Rent/deadline calculators
  • /tools/generators.html - Notice/letter generators

Help Sub-Pages:

  • /help/faq.html - Knowledge base
  • /help/crisis.html - Emergency resources (tenant)
  • /help/contact.html - Contact forms
