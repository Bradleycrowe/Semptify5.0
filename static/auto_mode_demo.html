      🚀 How to Use Auto Mode

      Step 1: Enable Auto Mode

      POST /api/auto-mode/toggle { "enabled": true }

      Step 2: Configure Features

      POST /api/auto-mode/config { "auto_generate_timeline": true, "auto_generate_calendar": true, "auto_identify_complaints": true, "auto_assess_rights": true, "auto_detect_missteps": true, "auto_suggest_tactics": true }

      Step 3: Batch Analyze Existing Documents

      POST /api/auto-mode/batch-analysis?limit=10

      Or Upload New Document (Auto Mode Triggered)

      POST /api/intake/upload Content-Type: multipart/form-data file:

      View Results Dashboard

      GET /static/auto_analysis_summary.html GET /static/batch_analysis_results.html

      🏗️ System Architecture

      Document Upload ↓ DocumentFlowOrchestrator (if auto_mode enabled) ├─ TimelineBuilder → Extract events & dates ├─ CalendarService → Create calendar events ├─ LegalAnalysisEngine → Assess rights & missteps ├─ ComplaintWizard → Identify complaint agencies ├─ ProactiveTacticsEngine → Suggest defense tactics └─ AutoModeSummaryService → Generate summary ↓ Results Published via Event Bus ↓ Real-time UI Update ├─ Progress indicator (0-100%) ├─ Confidence score display ├─ Urgent alerts ├─ Recommended actions └─ Next steps workflow

      ✨ Key Features

      • ✅ Timeline Extraction - Automatically extract dates, events, and deadlines from documents
      • ✅ Calendar Generation - Create calendar events with reminders for critical dates
      • ✅ Complaint Identification - Identify regulatory agencies where you can file complaints
      • ✅ Rights Assessment - Evaluate your legal rights and tenant protections
      • ✅ Missteps Detection - Identify legal procedural violations by landlord
      • ✅ Tactics Recommendation - Suggest proactive defense strategies
      • ✅ Smart Summaries - Generate comprehensive analysis summaries
      • ✅ Actionable Recommendations - Prioritized action items with time estimates
      • ✅ Batch Processing - Analyze multiple documents simultaneously
      • ✅ Progress Tracking - Real-time progress indicators and confidence scores
